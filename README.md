# HRV Weekly Tracker

Personal HRV optimization tracker with daily checklists, exercise weight tracking, and trend charts.

## Setup (GitHub Pages)

1. Create a new repository on GitHub (e.g., `hrv-tracker`)
2. Upload all files from this folder to the repo
3. Go to **Settings → Pages → Source → Deploy from branch → main → / (root) → Save**
4. Your tracker will be live at `https://yourusername.github.io/hrv-tracker`
5. On your iPhone: open the URL in Safari → Share → Add to Home Screen

## Data

All data is stored in your browser's localStorage. It persists across sessions but is per-device and per-browser.
